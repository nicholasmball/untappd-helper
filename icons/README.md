# Extension Icons

This folder should contain PNG icons for the extension:
- `icon16.png` - 16x16 pixels (toolbar)
- `icon48.png` - 48x48 pixels (extensions page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Generate Icons

1. Open `generate-icons.html` in a browser
2. Click each download link to save the PNG files
3. Save them to this folder with the correct names

## Custom Icons

Replace these with custom icons if desired. Icons should:
- Be PNG format
- Have transparent backgrounds (optional)
- Be the exact sizes specified above
